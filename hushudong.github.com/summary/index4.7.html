今日所学总结

(一)上午所学总结

  • 1.今天老师带我们回顾呢昨天所学的HTML DOM方法；
  • 2、a.创建：document.createElement(元素名称/节点名称)。
  • 3、b.增加：document.inserBefore(父元素，要往父元素插入的元素)。
  • 4、c:插入：document.appendChild()。
  • 5、d:删除：document.removeChild(Node对象)childNodes/children。
  • 6、e:修改：document.replaceChild(新元素，旧元素)或者叫移动。
  • 7、f：查找：document.getElementById/getElementByTagName/getElementClassName/and so on.....。
  • 8、接着继续上呢鼠标事件以及键盘输入事件等。
  • 9、冒泡型事件传播，从里面到外面的一层层传播；(一般很少用)。
  • 10、捕获型事件传播，从外面到里面的捕获到具体位置
  • 11、event.cancelBubble=true;是否取消冒泡。
  • 12.键盘输入顺序:37左，38上，39右，40下等。

(二)下午所学总结

  • 1.onabort表示图像的加载被中断。
  • 2.onblur表示元素失去焦点。
  • 3.onchange表示域的内容被改变。。
  • 4.onclick表示当用户点击某个对象时调用的事件句柄。
  • 5.onfocus表示元素获得焦点。
  • 6.onkeydown表示某个键盘按键被按下。。
  • 7.onmousedown表示鼠标按钮被按下。.
  • 8.onmousemove表示鼠标被移动。
  • 9.onmouseout表示鼠标从某元素移开。
  • 10.onselect表示文本被选中。.
  • 11.onsubmit表示用户退出页面。
